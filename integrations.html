EPAM Report Portal

Unicorn has ability to generate powerful test results report using EPAM Report Portal

Just deploy EPAM ReportPortal instance, add tests project dependency to Unicorn.Reporting.ReportPortal package and initialize reporter during tests assembly initialization.

See documentation to learn how to integrate your tests with ReportPortal.
